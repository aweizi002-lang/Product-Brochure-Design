# 产品简章折页设计工具

## 功能说明
在线设计产品简章折页，支持多种折页类型和设计风格，一键导出PDF。

## 支持的折页类型
| 类型 | 尺寸 | 页面数 |
|------|------|--------|
| 二折页 | 360mm × 285mm | 4面 |
| 三折页 | 540mm × 285mm | 6面 |
| 四折页 | 720mm × 285mm | 8面 |

## 设计风格
- 🟠 橙色科技风 - 适合科技产品、数字化解决方案
- 🔴 红色商务风 - 适合销售培训、企业内训
- 🔵 蓝色商务风 - 适合管理课程、领导力培训
- 🟢 绿色清新风 - 适合健康、环保、教育类
- 🟣 紫色高端风 - 适合高端课程、总裁班
- 🟡 金色尊贵风 - 适合总裁班、高端研修

## 内容模块
1. **封面** - 产品名称、Slogan、核心卖点
2. **痛点页** - 企业常见问题
3. **解决方案** - 功能模块介绍
4. **课程安排** - 时间、地点、课程表
5. **讲师介绍** - 照片、头衔、经历
6. **学习指南** - 适合对象、报名方式、二维码

## 部署步骤

### Render部署
1. 上传代码到GitHub
2. 在Render创建Web Service
3. 配置：
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `streamlit run app.py --server.port $PORT --server.address 0.0.0.0`

### 本地运行
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 使用流程
1. 选择折页类型和设计风格
2. 依次填写各面内容
3. 上传需要的图片（讲师照片、二维码等）
4. 预览效果
5. 导出PDF

## 注意事项
- 当前v1版本为基础框架，PDF生成功能开发中
- 字体文件需自行上传到服务器（HarmonyOS Sans SC）
